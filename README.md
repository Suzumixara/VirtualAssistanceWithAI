# 🎙️ Виртуальный голосовой ассистент (Python + AI)

Проект представляет собой голосового ассистента с графическим интерфейсом, способного:

- распознавать речь 🎤  
- озвучивать ответы 🔊  
- выполнять системные команды 💻  
- обращаться к ИИ через API 🤖  
- работать как в голосовом, так и в текстовом режиме  

---

## 🚀 Возможности

### 🧠 Основные функции
- Голосовая активация по слову **"помощник"**
- Распознавание речи (SpeechRecognition)
- Озвучивание ответов (pyttsx3)
- Интеграция с AI (Hugging Face / DeepSeek)
- Обработка текстовых команд через GUI

### ⚙️ Системные команды
Ассистент умеет:
- Открывать программы (браузер, блокнот, Discord, Steam и др.)
- Показывать погоду
- Сообщать текущее время
- Выключать компьютер

### 🖥️ GUI (PyQt5)
- Лог диалога
- Ручной ввод команд
- Кнопки управления (старт / стоп)
- Системный трей

---

---

### ⚡ Как это работает
1. Микрофон → распознавание речи
2. Проверка системных команд
3. Если команда не найдена → запрос к AI API
4. Ответ → текст + озвучка
