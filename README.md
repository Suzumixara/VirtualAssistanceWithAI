# 🎙️ Виртуальный голосовой ассистент (Python + AI)

Проект представляет собой голосового ассистента с графическим интерфейсом, способного:

- распознавать речь 🎤  
- озвучивать ответы 🔊  
- выполнять системные команды 💻  
- обращаться к ИИ через API 🤖  
- работать как в голосовом, так и в текстовом режиме  

---

## 🚀 Возможности

### 🧠 Основные функции
- Голосовая активация по слову **"помощник"**
- Распознавание речи (SpeechRecognition)
- Озвучивание ответов (pyttsx3)
- Интеграция с AI (Hugging Face / DeepSeek)
- Обработка текстовых команд через GUI

### ⚙️ Системные команды
Ассистент умеет:
- Открывать программы (браузер, блокнот, Discord, Steam и др.)
- Показывать погоду
- Сообщать текущее время
- Выключать компьютер

### 🖥️ GUI (PyQt5)
- Лог диалога
- Ручной ввод команд
- Кнопки управления (старт / стоп)
- Системный трей

## 🛠️ Установка

### 1. Клонировать репозиторий
```bash
git clone https://github.com/your-username/voice-assistant.git
cd voice-assistant
2. Установить зависимости
pip install -r requirements.txt
🔑 Настройка API
Hugging Face

В файле VoiceAssistantCore:

self.HF_API_TOKEN = "ВАШ_API_КЛЮЧ"

Получить ключ: https://huggingface.co/settings/tokens

DeepSeek (если используешь)
DEEPSEEK_API_KEY = "ВАШ_API_КЛЮЧ"
▶️ Запуск
GUI версия:
python main.py
Консольная версия:
python voice_assistant.py
🎤 Примеры команд

Скажи:

"помощник открой браузер"
"помощник сколько времени"
"помощник открой стим"
"помощник какая погода"
"помощник расскажи анекдот"
⚡ Как это работает
Микрофон → распознавание речи
Проверка системных команд
Если команда не найдена → запрос к AI API
Ответ → текст + озвучка
🔒 Безопасность
Не публикуй API ключи в открытом доступе
Добавь .env или конфиг-файл для ключей
🧩 Возможные улучшения
🔊 Добавить wake-word detection (например, Porcupine)
🌍 Поддержка нескольких языков
🧠 Локальная модель (например, через Ollama)
🪟 Автозапуск при старте системы
📱 Интеграция с Telegram / мобильным приложением
🐞 Известные проблемы
PyAudio может сложно устанавливаться
Распознавание речи зависит от качества микрофона
Иногда API может возвращать ошибки (нужно обрабатывать)
