Голосовой помощник на Python 

Данный проект представляет собой голосового ассистента с графическим интерфейсом, разработанного на языке Python. Ассистент способен распознавать голосовые команды пользователя, выполнять системные операции, отвечать на вопросы с помощью искусственного интеллекта, а также озвучивать свои ответы.

Основная логика работы ассистента разделена на три файла. В файле test2.py содержится класс VoiceAssistantCore, который отвечает за распознавание речи через микрофон с использованием библиотеки speech_recognition, синтез речи через pyttsx3, выполнение системных команд (открытие браузера, блокнота, Steam, Discord, проводника, калькулятора, показ текущего времени, погоды в браузере и выключение компьютера), а также отправку запросов к API Hugging Face для получения ответов от нейросети. Для поиска путей к установленным программам используется работа с реестром Windows.

Файл test1.py реализует графический интерфейс на PyQt5. Пользователю доступно окно с логом сообщений, полем для ручного ввода текстовых запросов и кнопками для запуска и остановки голосового режима. Ассистент работает в отдельном потоке, чтобы не блокировать интерфейс. Также реализована поддержка системного трея, позволяющая свернуть приложение в фоновый режим. Все ответы ассистента отображаются в логе и одновременно озвучиваются через динамики.

Файл Asistant.py представляет собой консольную версию ассистента без графического интерфейса, работающую исключительно через командную строку. В этой версии также присутствует интеграция с DeepSeek API в качестве альтернативного источника ответов от ИИ.
